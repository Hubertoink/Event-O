A clean event management plugin with Gutenberg blocks (list/accordion + carousel).

== Description ==

Event_O provides:
- Custom Post Type: Event
- Taxonomies: Event Category, Venue, Organizer
- Simple event fields: start/end datetime + price + status
- Gutenberg blocks (dynamic): Event List (Accordion) + Event Carousel
- Optional single template (toggle in Settings → Event_O)

== Installation ==

1. Copy the `event-o` folder into `wp-content/plugins/`.
2. Activate “Event_O” in WP Admin → Plugins.
3. Add events in WP Admin → Event-O.
4. Use blocks “Event_O – Event List” and “Event_O – Event Carousel”.
